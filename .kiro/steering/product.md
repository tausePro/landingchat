# LandingChat - Product Overview

LandingChat is a **conversational commerce platform** that enables businesses to create AI-powered storefronts with chat-based shopping experiences.

## Core Value Proposition
- Multi-tenant SaaS platform where organizations create their own branded stores
- AI-powered chat agents (using Claude/Anthropic) assist customers with product discovery and purchases
- Subdomain-based store access (e.g., `store-name.landingchat.co`)

## Key Features
- **Storefront Builder**: Customizable templates (minimal, services, single-product, complete)
- **Product Management**: Inventory, variants, subscriptions, configurable options
- **AI Chat Agent**: Conversational shopping assistant per store
- **Customer Management**: Customer identification, order history
- **Marketing Tools**: Badges, promotions, coupons, shipping rules
- **Dashboard**: Organization management, analytics, settings

## User Roles
1. **Platform Admin**: Manages organizations, users, marketplace, system settings
2. **Organization Owner**: Manages their store, products, agents, customers, orders
3. **Store Customer**: Browses products, chats with AI agent, makes purchases

## Business Model
- Organizations (tenants) subscribe to the platform
- Each organization gets a subdomain storefront
- Customers interact via chat or traditional browsing
